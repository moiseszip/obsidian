## Objeto
- **atributos** → propriedades
- métodos → funcionalidades

		//criando 
		const celular = { 
			cor: 'preto', 
			ligar: function() { 
			alert("Ligando") 
			} 
		} 
		//usando 
		celular.ligar()	
		
		//output 
		"Ligando"
